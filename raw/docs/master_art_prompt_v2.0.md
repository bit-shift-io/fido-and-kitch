# Master Art Asset Prompt (V2.0)

Copy and use this template when generating individual game assets or level mockups for your 2D multi-level platformer.

```text
[Specific Asset or Level Description, e.g., Multi-tiered stone archway platform with wooden climbing ladders and copper steampunk gears]

-- Perspective & Layout: 2D side-view multi-level platformer perspective (Lurid Land / retro puzzle-platformer layout with verticality, platforms, and climbing ladders).
-- Style (Foreground Gameplay Elements): Crisp 2D vector cartoon style, thick dark rounded outlines, flat vibrant fills, clean graphic shapes, inspired by Bluey cartoon style, kid-friendly and highly legible.
-- Foreground Palette: Bold, saturated primary and secondary colors (e.g., ruby red gems, vibrant turquoise water, golden brass gears, rich mossy greens). Zero blur, high contrast for instant player readability.
-- Style (Midground Environment Layer): Detailed Van Gogh painterly post-impressionist style with clear textures and rich color saturation. Features medieval castle structures, stone bridges, and near fields.
-- Style (Far Background Layer): Rich, swirling Van Gogh night sky with starry impasto brushstrokes, glowing moon/sun, and distant mountains.
-- Depth Strategy (3-Layer Parallax): Clear separation between crisp vector foreground, structured painterly midground, and sweeping swirling far background. Maintains full visual clarity without applying heavy depth-of-field blur.
-- Negative Prompt: Photorealistic 3D render, dark gritty fantasy, messy sketch lines, blurry foreground, washed-out muted pastel watercolor, pixel art.
```
